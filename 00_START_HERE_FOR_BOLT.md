# HMG Newsroom — Bolt Thin UI Surgery Source

This is a thin, targeted source branch for Bolt.

Do not create a new unrelated app.
Do not build a landing page.
Do not make docs the main output.
Do not publish.

The full app exists in Replit/Codex. This branch is intentionally thin so Bolt does not choke on context.

Mission:
Make HMG Newsroom feel Apple-ready, urgent, simple, premium, and powerful.

Product law:
If the app does not restore the user’s last screen, it must open to HMG Quick Launch — not Founder Command Center, not stats, not backend health, not a storage-bin monitor board.

Default home must prioritize:
- Haven brand selector
- Breaking News
- Article Draft
- Photo / WebArt
- Video / WebEdit
- Social Pack
- WordPress Draft / Publish
- Resume Last Draft

Core upgrades:
- Quick Launch home
- Editorial Desk simplified into field-publishing flow
- WebArt as Photoshop-light
- WebEdit as Opus Clip-light
- WordPress post-from-anywhere
- visual command menu
- theme consistency
- backend/API response safety
- autosave/output history
- OG journalist ease-of-use

Work in small tasks. Do not attempt every task in one prompt.
